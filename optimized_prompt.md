# 最佳化提示詞

請扮演一位熟悉國小自然科教學、互動式網頁設計、Python 靜態網站產生與 GitHub Pages 發佈的資深工程師兼教材設計師。根據使用者提供的 PDF「114國小自然6下課本PDF_第1單元.pdf」中，單元 1「巧妙的施力工具」活動 2「滑輪與輪軸」的教材內容，完成一個可公開瀏覽的互動式教學網頁。

工作要求：

1. 先從 PDF 中辨識活動 2「滑輪與輪軸」的重點，不逐字大量複製教材，不嵌入課本原圖，而是用適合國小六年級學生的語言重新整理。
2. 內容需包含滑輪與輪軸兩大主題：
   - 定滑輪：不省力也不費力，會改變施力方向。
   - 動滑輪：可以省力，手的移動方向與物體移動方向相同，但需考慮滑輪本身重量。
   - 輪軸：輪與軸一起轉動，是槓桿原理的應用；施力在輪上通常省力，施力在軸上通常費力。
   - 生活應用：吊車、門把、水龍頭、吊扇、竹蜻蜓、螺絲起子等。
3. 寫一個不需外部套件的 Python 程式，產生單一 `index.html`。
4. `index.html` 必須是完整可用的互動式教學頁，包含：
   - 滑輪模擬器：可切換定滑輪/動滑輪、調整重物重量、即時計算施力大小與方向。
   - 輪軸模擬器：可調整輪半徑、軸半徑、重物重量，並切換施力在輪或施力在軸。
   - 可視化圖解：使用 HTML/CSS/SVG，不依賴外部圖片或 CDN。
   - 實驗紀錄或統計比較區。
   - 生活應用判斷題與即時回饋小測驗。
   - 響應式版面，手機與桌機皆可讀。
5. 風格需適合國小高年級自然課：清楚、活潑但不幼稚；介面要像教學工具，不做行銷式 landing page。
6. 完成後執行或等效產出 `index.html`，檢查 HTML/JS 語法與主要互動元素存在。
7. 若 GitHub 工具與憑證可用，建立 GitHub repository，啟用 GitHub Pages，並回傳可直接開啟的公開網址。若環境缺少 GitHub CLI、git 或登入憑證，需清楚說明限制並保留可發布的網站檔案。

輸出成果：

- `build_interactive_lesson.py`
- `index.html`
- GitHub Pages 網址，或明確的發佈阻礙與下一步
